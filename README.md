# ⏱ FocusFlow — Smart Productivity Timer

A beautiful Pomodoro-based productivity app with task management, animated progress rings, and weekly statistics — designed to help you stay focused.

## ✨ Features

- **🍅 Pomodoro Timer** — 25min focus / 5min short break / 15min long break
- **🔄 Animated Progress Ring** — SVG ring with gradient stroke showing remaining time
- **✅ Task Manager** — Add, complete, and delete daily tasks
- **📊 Weekly Stats** — Focus time, sessions, tasks done, and day streak
- **📈 Activity Chart** — Visual bar chart of daily sessions
- **💬 Motivational Quotes** — Random inspirational quotes to keep you going
- **🔔 Sound Notification** — Audio alert when timer completes
- **💾 Persistent Data** — All progress saved to localStorage
